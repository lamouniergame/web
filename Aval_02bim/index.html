O Preistexu Customer Manager é um sistema desenvolvido pela Preistexu Enterprises. O sistema foi desenvolvido para integrar o gerenciamento de empresas que lidam com diversos clientes e fornecedores.

Esse sistema permite criar uma lista com o seus clientes e fornecedores, além de permitir editá-los, excluí-los e organizá-los.
